About

This project demonstrates the usage of quarkus with the qute templating engine and hotwire turbo. It consists of examples demonstrating how templates can be split, partially loaded and also streamed via SSE. The following parts are used in this example:

  • Quarkus
  • Quarkus - Qute
  • Hotwire - Turbo
  • Creative Tim - Paper Dashboard 2
